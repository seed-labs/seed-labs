# BGP RPKI and Route Origin Validation Lab

This folder contains a SEED-style lab description for teaching RPKI and Route
Origin Validation (ROV) using the SEED Internet Emulator.

Files:

- `BGP_RPKI_ROV.tex`: lab description in LaTeX format
- `Makefile`: compiles the lab description into PDF
- `Labsetup/`: helper files for ROA tables and BIRD filter generation

To build the PDF on a machine with LaTeX installed, run:

```bash
make
```
