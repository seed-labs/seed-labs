# BGP Prefix Hijacking Lab

This folder contains a standalone SEED-style lab for BGP prefix hijacking using
the SEED Internet Emulator.

Files:

- `BGP_Prefix_Hijacking.tex`: lab description in LaTeX format
- `Makefile`: compiles the lab description into PDF
- `Labsetup/`: helper snippets for BIRD static routes and filters

To build the PDF on a machine with LaTeX installed, run:

```bash
make
```
